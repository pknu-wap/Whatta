# WHATTA – 에이전트형 통합 일정 관리 앱

**왓타는 단순한 일정 기록 앱을 넘어,**  
**사용자의 하루를 이해하고 먼저 관리해주는 에이전트형 일정 관리 앱을 목표로 합니다.**  

<br>
  
> “다음 일정이 뭐더라?”,
> 
> "이건 또 언제 하지?",
> 
> “언제 나갈까”,
> 
> "지각하진 않겠지?"

👉🏻 이와 같은 고민을 반복하는 현대인을 위한 **에이전트형 통합 일정 관리 앱**입니다.

<br>

## ⚡ 현재 핵심 기능
| 기능 | 설명 |
|------|------|
| 📷 **OCR 기반 일정 등록** | 시간표 이미지를 업로드하면 일정 자동 생성 (에타/학교 포털 지원) |
| 🚗 **교통 알림 기능** | 설정한 시간에 맞춰 대중교통 도착 예정 정보 알림 제공 |
| 👆🏻 **드래그 앤 드롭 스케줄링** | 일정을 직접 이동 및 재배치, 직관적인 일정 관리 가능 |

<br>

👉🏻 이처럼, 지금까지의 왓타는

사용자가 **일정을 입력하고 관리하는 과정을 최대한 간편하게 만드는 것**에 집중해 왔습니다.

<br>

## 🚀 Whatta의 새로운 비전

현재의 왓타를 포함한 대부분의 캘린더 앱은

👉🏻 **직접 계획을 세우는 사람에게 유리한 구조**입니다.

<br>

하지만 현실에서는 많은 사람들이

> - 일정이 생길 때마다 바로 정리할 여유가 없고
> - 여러 일정이 동시에 쌓이면서 우선순위를 판단하기 어려우며
> - 결국 계획 없이 그때그때 처리하거나  
> - 마감일에 임박해서 몰아서 해결하게 됩니다
> 
>   또한, 일정 관리 자체가 하나의 ‘작업’이 되기 때문에  
바쁜 상황에서는 앱에 기록하는 것조차 부담으로 느껴집니다. 

<br>

👉🏻 즉, **“정리를 잘하는 사람에게는 유용하지만  
정말 정리가 필요한 사람에게는 잘 쓰이지 않는 앱”**
이라는 한계가 있습니다.

<br>

왓타는 이렇게 생각했습니다.

<br>

**“일정을 기록하기만 하는 앱이 아니라,  
일정을 함께 관리 앱이 되자.”** 

<br>

그래서 우리 왓타는

👉🏻 **에이전트형 통합 일정 관리 앱**로 나아가려 합니다.

<br>

## ⚡ 추가될 핵심 기능

| 기능 | 설명 |
|------|------|
| 🤖 **AI 기반 일정 생성** | 자연어 입력 (“내일 3시에 회의 추가해줘”) 또는 이미지(대화, 과제 목록 등)를 통해 일정 자동 생성 |
| 🧠 **자동 스케줄링** | 빈 시간과 사용자 패턴을 분석하여 최적의 시간대에 자동 배치 |
| 🚗 **능동형 교통 알림** | 출발지/도착지 기반 경로 추천 및 출발 시간 자동 계산 + 실시간 교통 정보 제공 (Live Activity 지원) |
| 🔗 **외부 서비스 연동** | Google Calendar, Notion DB 연동 |
| ⚙️ **DnD 기능 고도화** | 드래그 앤 드롭 기반 일정 관리 및 스케줄링 기능 확장 |

<br>

## ⚙️ Tech Stack

### 📱 App
- React Native (Expo)
- React Navigation
- Axios

### 👾 Backend
- Spring Boot (Java 17)
- MongoDB
- JWT Authentication
- Clova OCR API
- Firebase Cloud messaging (FCM)

### ☁️ DevOps / Infra
- Docker
- GitHub Actions (CI/CD)
- Google Cloud Run (GCP)
- Swagger UI
