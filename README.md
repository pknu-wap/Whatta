# 🗓️ Whatta – 통합 일정 & 할 일 관리 앱

**💡 Whatta(왓타)는 단순한 일정 관리 앱을 넘어,**  
**사용자의 하루를 이해하고 먼저 챙겨주는 비서 앱을 목표로 합니다.**  
  
  
> “이제 뭐 해야 하지?”, “언제 나가야 하지?”  

와 같은 질문을 하루에도 여러 번 되뇌는 현대인들을 위한 비서형 일정 관리 앱입니다.
  
바쁜 일상 속 우리는 시간표·알바·과제·약속 등을 각각 다른 앱에서 확인해야 합니다.
하지만 이러한 정보의 분산은 일정 중복이나 누락, 비효율적인 시간 사용으로 이어집니다.

Whatta는 이런 문제를 해결하기 위해,  
📍 시간표와 근무표 같은 고정 일정, 약속·과제 같은 가변 일정(Task)을 **하나의 통합 UI에서 관리**할 수 있도록 설계되었습니다.  
📍 또한, 단순히 리마인드 알림뿐만 아니라 **교통 정보·출발 알림**까지 제공합니다.  
  

이외에도,

📍 **OCR 기반 자동 등록**으로 시간표 이미지를 올리면 일정이 자동 생성되고,

📍 **드래그 앤 드롭** 캘린더로 별도의 설정창 없이 시각적으로 스케줄링할 수 있습니다,  

    
## 🚀 Core Features

| 기능 | 설명 |
|------|------|
| 🗓️ **Calendar View** | 일간·주간·월간 단위로 일정을 한눈에 확인 |
| ✅ **Task Management** | 체크박스 기반 할 일 관리, 라벨·반복설정 지원 |
| 📷 **OCR Timetable Import** | 시간표 이미지 자동 인식 및 일정 등록 |
| 🏷️ **Label System** | 카테고리별 라벨 지정 및 필터링 지원 |
| 🔔 **Push Notification** | 일정 시작 전 알림 및 루틴 알림 제공 |

## ⚙️ Tech Stack

### 📱 App
- React Native (Expo)
- React Navigation
- Axios

### 👾 Backend
- Spring Boot (Java 17)
- MongoDB (Atlas)
- JWT Authentication
- Cloud Run (GCP)
- Clova OCR API

### ☁️ DevOps / Infra
- Docker
- GitHub Actions (CI/CD) (🚀구현 예정)
- GCP Cloud Build (🚀구현 예정)
- Swagger UI
